/r/skyrimmods and /r/skyrim - these subreddits are great.

File types used in Skyrim Modding (WIP)

.esm files are Master Files. Big mods need these, like Falskaar. Otherwise, most mods are not .esm files.

.esp files are Plugin Files. These are mainly the files for mod plugins.

.bsa are "bethesda archives" and can be unpacked using a .bsa unpacker tool.

.seq files were used (or are still being used) to address a mod's custom voice/dialogue bug.
