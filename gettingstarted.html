Getting Starting

The "Getting started" section is aimed at people that are not familiar with modding Skyrim or other Bethesda games (such as Fallout: NV, or Oblivion, for example). Most mods require certain other mods, like SKSE for giving mods improved scripting capabilities, or SkyUI for the Mod Configuration Menu (MCM).

The mods listed on this page do not change gameplay mechanics or graphics. Most of the items listed here are utilities and tools to let Skyrim mods have more features and work correctly.

First of all, if you just redownloaded Skyrim, make sure you start the game first before modding. On the launcher, select the Ultra graphics setting, then start the game. Now exit Skyrim. This is to insure that the ini files are generated correctly.

Note: If you don't like the Skyrim Nexus, most of the mods in this post are also available on the Steam Workshop.

Tools/Utilities listed here:

  • Skyrim Script Extender - SKSE
  • Better Oblivion Sorting Software - BOSS
  • Nexus Mod Manager - NMM
  • SkyUI
  • TES5Edit
  • WryeBash
  • Skyrim.ini Tweaks
  • Skyrimprefs.ini Tweaks

Skyrim Script Extender (SKSE)

Next, you will need Skyrim Script Extender (SKSE). Download SKSE here http://skse.silverlock.org/. You will now need to start Skyrim with skse_launcher.exe in order for SKSE, and mods that require it, to work. Personally, I make a shortcut to skse_launcher.exe then pin that to the start menu. You will know if SKSE is working properly if you have a SKSE message in the top-left of the screen when starting Skyrim, as shown in this screenshot.

Better Oblivion Sorting Software (BOSS)

Then, get BOSS - to keep your mods properly "load-ordered", to improve performance and reduce CTDs due to improper mod load order. Install your mods with NMM, then after that run BOSS to sort the mods. However, do pay attention to each mod that you do download to see if it needs to be loaded last, or above/below another mod, as the BOSS masterlist of mods is updated on a regular basis but is not 100% precise.

If you are interested in trying a newer version of BOSS, check out LOOT which stands for: Load Order Optimisation Tool.

Nexus Mod Manager (NMM)

Now, to easily install most of these mods, get the Nexus Mod Manager (NMM). With NMM, you can download mods with a click of a button from the Skyrim Nexus website, then install/uninstall mods with the NMM program, as well as configure your load order. Not required since you can always install mods manually, but NMM is highly recommended because of the automated installation process.

SkyUI

SkyUI is an absolute must have. The changes to the UI are keyboard and mouse friendly, and has excellent controller support for those that play with a controller. Also included in SkyUI is the amazing Mod Configuration Menu, where you can change mods settings in game (as long as the mod has MCM support, of course).

TES5Edit

TES5Edit is a tool that I have not used extensively, but mod creator's do use TES5Edit to remove bad data/record changes they made by accident, and to "clean" their mod. More documentation is here: http://www.creationkit.com/Category:TES5Edit.

WryeBash

Wrye Bash is a tool that was used heavily in modding Oblivion. You can still use it to be a mod installer/organizer of sorts, but it's main feature we want to use is the Bashed Patch. With the Bashed Patch we can merge mods leveled lists together. With this feature, you'll be able to see the loot from both Immersive Armors and Immersive Weapons in the leveled list. If you do not create a Bashed Patch, but install a several mods that use leveled lists, only the last mod loaded in the load order will influence the leveled list.

TLDR; Create a Bashed Patch with Wrye Bash so you get to see items in different mods leveled lists in game (ie: merge leveled lists).

Steps to create a Bashed Patch with Wrye Bash:
  • Scroll down the mod list and right-click Bashed Patch, 0.esp and select Rebuild Patch.
  • A popup asking to deactive mods prior to patching may appear. If it does, simply click [OK]. This will deactivate said mod file and it will be merged into the bashed patch. Check only "Merge Patches", "Import Names", "Import Stats", "Tweak Settings", and "Leveled Lists" in the left pane.
  • Click on "Tweak Settings" in the left pane.
  • Click on "Leveled Lists" in the left pane.
  • Click [Build Patch] at the bottom of the window. A dialogue asking you to activate it may appear, click [Yes].

The Bashed Patch, 0.esp is now generated into your skyrim/data folder, and should always be loaded last!

Skyrim.ini Tweaks

Change these the Skyrim.ini file with a text editing program (such as Notepad) to tweak certain settings. The text in brackets (eg [General]) is that section's header.

Disable the intro video/logo splash. - Shave off a few seconds when Starting up Skyrim! Either delete the intro.bik video file in Skyrim's Skyrim/data/video folder, or you can do this .ini tweak in the [General] section.

Recommended 
[General]
sIntroSequence=

Field of View (FoV) - Increase your FoV with this tweak. Default is 65, IIRC. I usually rock 70-75 degrees because I mainly play in 3rd person, and sit farther back. If you like to play in first person more often, you will probably want to increase your FoV to 75 degrees or more.

If you'd like, you can learn more about FoV here and here.

Recommended 
[Display]

fDefault1stPersonFOV=75.0
fDefaultWorldFOV=75.0

Ranged/Magic Killcam Tweak - Enables a much higher frequency of ranged or magic killcams.

Optional 
[VATS]
fVATSCastingAfterKillDelay=1.2000
uVATSRangedPercentSneak=100
uVATSRangedPercentGlobal=100
bVATSIgnoreProjectileTest=0
bVATSAllowNoKill=0
bVATSMultipleCombatants=1
bVATSForceRanged=1

Disable all killcams - Tired of killcams or think they break immersion? Disable them with this tweak!

Optional 
[VATS]
bVATSDisable=1

Shadow Flicker - A viewer of mine told me about this great tweak. This tweak will fix shadow flickering caused by the sun when it updates its position.

Optional 
[Display]
fSunShadowUpdateTime=0
fSunUpdateThreshold=2.0

Torch/Light Source Pop-In - Enable a higher distance for the drawing of torches/light sources. Default in the game is 1024, which is pretty close.

Optional 
[General]
fFlickeringLightDistance=4096

SkyrimPrefs.ini

Change these the Skyrim.ini file with a text editing program (such as Notepad) to tweak certain settings. The text in brackets (eg [General]) is that section's header.

SkyrimPrefs.ini has parameters that control other configuration options in Skyrim. Much like editing Skyrim.ini in the section before, these tweaks are optional.

bFloatPointRenderTarget - for ENB users - This tweak is required for ENBs to properly use floating point rendering which allows the cooler graphic effects, such as SSAO, DoF and more. This setting will cause a performance hit.

Enable if using ENB 
[Display]
bFloatPointRenderTarget=1

Shadow Map Resolution - Bit of a tweak to reduce the exessive iShadowMapResolution that is set to 4096 by the Ultra graphic setting. You lose hardly any shadow graphical quality, but gain a big increase in performance.

Strongly recommended, performance gain 
[Display]
iShadowMapResolution=2048

Shadows Tweaks - Enable shadows that trees and land/rocks cast. Small performance hit, but increases graphical quality.

Optional, small performance hit 
[Display]
bDrawLandShadows=1
bTreesReceiveShadows=1

Shadow and Grass Distance Fade - This tweak pushes back the distance for grass and shadows to fade. Minor performance hit.

Optional, small performance hit 
[Display]
fShadowLODStartFade=400.0000

[Grass]
fGrassStartFadeDistance=18000

Disable Mouse Acceleration - Simply disables mouse acceleration.

Optional 
[Controls]
bMouseAcceleration=0

Misc. Information

Curious about what types of files (.esp. .bsa, .dds, etc) you're seeing/using? Check out http://wiki.step-project.com/Guide:Skyrim_Installation#Mod_Filetypes_and_Directories.

Bad performance?

Disable HD texture mods if your video card does not have much VRAM. Another thing to decrease are shadow settings, then resolution. Both of these are the main hogs of performance. Also, if you use an ENB, you may want to tone the ENB graphical effects down, or turn them off completely.
